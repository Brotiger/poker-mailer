# Расписной покер (Mailer)
Сервис рассылки писем

## Зависомости
* go1.22
* make

## Сборка и запуск
### Конфигурация
_Прописать в `.env` параметры и экспортировать:_
```bash
cp .env.example .env
source .env
```

### Make команды
```bash
make build # сборка проекта
make up # запуск скомпилированного проекта
make run # запуск без компиляции
```
